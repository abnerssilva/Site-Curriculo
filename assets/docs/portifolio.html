Projeto Unes

Iniciei com o projeto UNES (Uma Universidade Fictícia) para praticar conceitos básicos do HTML5, nele não uso nenhuma outra ferramenta como CSS3 ou JS para aplicar quaisquer efeitos. Toda a estrutura foi desenvolvida usando puramente o HTML e suas tags.

Projeto anna bella

Continuei o desenvolvimento dos estudos com o projeto Anna Bella (Modelo Fotográfica Fictícia) para iniciar a prática dos conceitos do CSS3, e das boas práticas de documentação de código. As estruturas utilizadas foram HTML5 (um pouco mais avançado do que no projeto anterior) e CSS3 simples, aplicando somente os conceitos de formatação usando Seletores, Classes e IDs em arquivo externo, "limpando" assim, o visual do código principal.

projeto tecblog

Dando continuidade, embarquei no projeto TecBlog (Blog de Tecnologia Fictício) para avançar nos conceitos e na aplicabilidade do CSS3. Aqui já podemos ver conceitos de Elementos Flutuantes inline e block, posicionamento estático, relativo, fixo e absoluto, efeitos em hyperlinks e heranças e especificidades.

projeto notícias cidade

Novamente, embarquei num projeto desafiador para um modelo de site de notícias não-responsivo, o Notícias Cidade (Site de Notícias Fictício) para avançar ainda mais nos conceitos e na aplicabilidade das stacks HTML5 e CSS3. Aqui já podemos ver criação de menus (superior e lateral), navegação com abas, layout fixo, imagens líquidas, colunas falsas e estilização de tabelas e formulários, além do início do uso do HTML semântico e recursos de acessibilidade para leitores de tela.

projeto chalé hotel

O projeto Chalé Hotel (Hotel Fictício) foi criado com o intuito de fortalecer os conceitos de HTML Semântico, boas práticas de escrita de código (comentários) e CSS visto até aqui, aliando com a nova técnica de layout líquido (ou fluido), onde o conteúdo do site se adapta para uma melhor experiência de visualização para o usuário. Ainda não se trata do conceito de responsividade, mas sim, de adaptabilidade para adequação dos frames às diversas telas até uma determinada faixa de corte.

projeto game mata mosquito

O Projeto Game Mata Mosquito foi o divisor de águas para eu entrar de cabeça no Vanila JS (JavaScript Puro), além disso, usei alguns conceitos básicos de jQuery integrado ao Bootstrap para adicioar responsividade e caixa de diálogo modal com efeitos. Foi um grande desafio para colocar meu aprendizado em prática. Além de toda a codificação JS, procurei continuar usando o HTML semântico e as boas práticas de escrita de código.

Pomodor + Lista de Tarefas

Esse Projeto foi concebido para entrar definitivamente no mundo real da contrução de aplicações que sejam 'utilizáveis' e sair do mundo fictício. Projeto conncebido do zero, com a construção da estrtura de marcação no HTML, do posicionamento e efeitos no CSS e a lógica desenvolvida em JS com aprofundamento na biblioteca jQuery. Além de toda a codificação lógica em JS/jQuery, procurei continuar usando o HTML semântico e as boas práticas de escrita e documentação do código. Será adicionado responsividade para adequação mobile futuramente.
